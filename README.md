# Troll v1.1.2
A simple plugin to troll player in Minecraft

Features:
- Freeze Player
- Open Player Inventory (InvSee)
- Kill Player
- Drop Player Hand Item
- Teleport to Player
- Fakeban
- Random Soundspawn
- Player control (Testing)
- Config (You can add any language)

Commands:
- /gethead player

Permissions:
- troll.opentrollmenu
